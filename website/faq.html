FAQ

Access to the node is granted until the end of an epoch. After that, you need to rent the node again.

If you validate through the Web App, you need access to a node in order attend the validation.

If you run a Desktop App (built-in node) and have stable internet, you can use that instead.

During the validation ceremony internet traffic peaks on nodes.

If your setup does not have a reliable internet connection you should rent a shared node.
